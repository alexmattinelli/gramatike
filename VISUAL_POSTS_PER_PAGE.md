## How It Works

### 1. Screen Width Detection
```javascript
function getPerPage() {
  const isMobile = window.innerWidth <= 980;
  return isMobile ? 3 : 10;
}
```

### 2. Dynamic API Call
```javascript
async function search(q, page = 1){
  const perPage = getPerPage();  // โ Gets current value
  const resp = await fetch(`...&per_page=${perPage}`);
  // ...
}
```
